# Webalkalmazás-fejlesztés 2026 - "ZH"

## Első lépések
1. Klónozzátok le a repository-t a saját gépetekre, például a `git clone https://github.com/attilagyongyosi/epam-nye-webapp-zh-2026.git` paranccsal.
2. Lépjetek be a projekt könyvtárába.
3. Telepítsétek fel NPM-mel a tesztfuttatáshoz szükséges package-eket az `npm install` vagy röviden `npm i` paranccsal.
4. Nézzétek meg hogy le tudjátok-e futtatni a teszteket (most még kapni fogtok elbukó eredményeket) az `npm test` paranccsal.

## Feladatok
Két lényeges feladat megoldását várjuk el:

### 1. HTML + CSS design
Navigáljatok el az `src/html-css/index.html` fájlhoz és kövessétek az ott lévő instrukciókat.

### 2. JavaScript
Navigáljatok el az `src/js/index.js` fájlhoz és kövessétek az ott lévő instrukciókat.
A kódbázis tartalmaz előre megírt teszteket az `src/js/index.test.js` fájlban, amikkel próbára tehetitek az implementációtokat.
Természetesen ezek jelenleg hibára fognak futni, amíg a megfelelő megoldást nem írjátok meg.

Ezeket a teszteket például parancssoron keresztül a gyökérkönyvtárban állva az `npm test` paranccsal tudjátok lefuttatni
illetve a VSCode és IntelliJ IDEA kínál nektek futtatási lehetőséget a teszt fájlon belülről is.

### Megoldások Beadása
Csináljatok 1 vagy több Git commitot a megoldásotokról és nyissatok egy új pull requestet GitHubon az eredeti repohoz.
